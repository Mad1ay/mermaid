# Source Code / Scripts

Цей каталог призначений для зберігання скриптів автоматизації та допоміжних інструментів для проєкту впровадження Odoo.

## Запланований функціонал

### Скрипти конвертації
- Конвертація документів з різних форматів
- Генерація діаграм Mermaid з текстових описів
- Експорт документації в різні формати (PDF, HTML тощо)

### Інструменти валідації
- Перевірка консистентності документації
- Валідація Mermaid діаграм
- Перевірка посилань між документами

### Автоматизація
- Автоматична генерація звітів
- Скрипти для оновлення статистики проєкту
- Інтеграція з системами контролю версій

## Структура (на майбутнє)

```
src/
├── converters/      # Скрипти конвертації
├── validators/      # Інструменти валідації
├── generators/      # Генератори документації
└── utils/           # Допоміжні утиліти
```
